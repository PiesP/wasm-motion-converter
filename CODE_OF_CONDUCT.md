# Code of Conduct

This project is a community space. We expect all participants to help keep it friendly, inclusive, and professional.

## Our standards

- Be respectful and constructive.
- Assume good intent; ask clarifying questions.
- Focus on the issue and the technical content.
- No harassment, hate speech, or discriminatory language.
- No sharing of private or sensitive data (yours or others').

## Scope

This applies to all repository spaces, including issues, pull requests, discussions, and code review.

## Enforcement

If you see unacceptable behavior, please report it.

- Security/privacy concerns: use the process in `.github/SECURITY.md`.
- Other concerns: open an issue asking for moderator attention (keep details minimal if sensitive).

Project maintainers may remove content or restrict participation if needed to keep the community safe and productive.
